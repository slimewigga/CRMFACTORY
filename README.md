# 💰 Деньги-Бабки CRM v2.0

CRM для AI Lab с базой данных, регистрацией и админ-панелью.

## Быстрый старт

1. Установи [Node.js](https://nodejs.org) (LTS)
2. Дважды кликни **`ЗАПУСК.bat`**
3. Откроется http://localhost:3000

**Первый вход:** `admin` / `admin123` — смени пароль в админке!

## Что нового

- **SQLite база данных** — все данные хранятся в `data/crm.db`
- **Регистрация и вход** — каждый участник со своим аккаунтом
- **Админ-панель** — управление пользователями, роли, блокировка, сброс пароля
- **Адаптивный дизайн** — телефон, планшет, большие мониторы (до 4K)

## Разделы CRM

| Раздел | Описание |

## Как пользоваться вдвоём

1. Первый (admin) заходит и настраивает CRM
2. Друг регистрируется через вкладку «Регистрация»
3. Оба видят одни и те же идеи, проекты и задачи
4. Распределяете задачи между собой в канбане

## Деплой в облако

Подробная инструкция: **[DEPLOY.md](DEPLOY.md)**

Кратко:
1. Залей проект на GitHub
2. Подключи к [Render.com](https://render.com) (файл `render.yaml` уже есть)
3. Добавь диск `/var/data` для сохранения базы
4. Задай `JWT_SECRET` в переменных окружения

После деплоя вы с другом заходите по одной ссылке из любого места.

## Уведомления

Все alert/confirm заменены на красивые toast и диалоги в стиле сайта.

## Запуск вручную

```bash
npm install
npm start
```

## Стек

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- БД: SQLite (better-sqlite3)
- Auth: JWT + bcrypt
